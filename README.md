# Tic Tac Toe

A Tic Tac Toe game built with HTML5, JS ES6, CSS, and using Jest for unit testing.

Demo available at https://davez.herokuapp.com/tic-tac-toe

## Required Software
- [ ] [Node.js](https://nodejs.org/en/download/)
- [ ] [NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)

## Running locally
- In a terminal
  - Navigate to the `tic-tac-toe` directory
  - Run `npx serve`
- Go to the local site displayed on the terminal
  - Most likely http://localhost:5000 

## Running unit tests
- In a terminal
  - Run `npm ci`
    - This will install Jest for testing and Babel to allow Jest to run with ES6 classes
  - Run `npm test`
